# To-Do List

[Деплой на GitHub Pages](https://thelonecheetah.github.io/web_lab2_to_do_list/)

## Задача
* Разработать веб-приложение To-Do-лист для создания, редактирования и удаляния задач, отметки выполнения, фильтрации (в том числе с помощью drag-and-drop), сортировки и поиска по дате и названию.
* Выполнить на чистом JavaScript с DOMAPI, HTML-файл должен содержать только подключение скрипта.
* Не использовать CSS-фреймворки (Bootstrap, Foundation и т.д.) и JS-фреймворков (Angular, React, Vue и т.д.).
* Предоставить deploy проекта.

## Реализованные функции
* Все элементы страницы создаются через JavaScript с использованием семантических HTML-тегов.
* Текст, атрибуты и стили применены через JavaScript, структура добавлена через методы DOM.
* Реализовано добавление, редактирование, удаления задач из списка.
* Доступна отметка задачи как выполненной с визуальным оформлением.
* Реализована фильтрация задач по статусу, а также поиск по названию.
* Обеспечено сохранение задач в localStorage и восстановление при загрузке страницы.
* Реализована сортировка задач по дате, а также ручное изменение порядка задач с помощью drag-and-drop.
